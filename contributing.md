## Easy steps:

* Clone this [repo](https://github.com/git-tips/tips.git):
  ```sh
  $ git clone https://github.com/git-tips/tips.git
  ```
  
* Install the deps: 
 ```sh
  $ cd tips && npm install
  ```
  
* Edit [tips.json](./tips.json) to add your tip in the below format:

```js
{
    "title": <your_tip_title>,
    "tip": <your_tip>
}
```

* Commit, push and send a PR!
