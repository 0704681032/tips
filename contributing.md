## Easy steps:

* [Fork](https://github.com/git-tips/tips/network) the repo.

* Install the deps:
 ```sh
  $ cd tips && npm install
  ```

* Edit [tips.json](./tips.json) to add your tip in the below format:

```js
{
    "title": <your_tip_title>,
    "tip": <your_tip>,
    "alternatives": [Optional list of alternatives]
}
```

* Commit, push and send a PR!
